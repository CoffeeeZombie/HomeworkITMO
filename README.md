# Лабораторные работы по программированию

Репозиторий содержит лабораторные работы по курсу программирования.

## Структура репозитория
- Каждая лабораторная работа выполняется в отдельной ветке
- Файл README также оформляется в отдельной ветке
- Для каждой ветки создается PR и после мержится в основную ветку

## Ветки
- `part-6` - лабораторные работа 6
- `Readme`, `ReadmeFix` - Для оформления README.md файла

## Технологии
- Язык: Python
- Система контроля версий: Git
